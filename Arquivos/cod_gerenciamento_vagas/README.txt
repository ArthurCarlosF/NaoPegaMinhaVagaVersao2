#  Parking Lot Detection OpenCV
  
  Base do projeto: 
  https://github.com/245charan/Parking-Lot-Detection-OpenCV
  
  
## Pré-requisitos:
    a. Python 3.7   
    b. OpenCV

## --------- Passos de execução sem o dashboard ------------
1. Executa o arquivo main.py
2. Demarca as vagas na figura aberta clicando na tela
3. Pressiona q

## --------- Passos de execução com o dashboard ------------
1. Executa o arquivo dashboard.py
2. Depura o arquivo main.py
2. Demarca as vagas na figura aberta clicando na tela
3. Pressiona q

É possível rodar sem a necessidade de demarcar as vagas. Nesse caso, basta remover as linhas 18 a 22,
do código em "main" e inserir as coordenadas das vagas no arquivo coordinates_1.yml, dentro da pasta "data".
